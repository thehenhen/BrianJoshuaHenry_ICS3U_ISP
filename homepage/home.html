What is this website?

Rather than re-inventing the wheel by creating yet another website that explains competitive programming concepts in detail, this website instead focuses on visualizing the concepts. For all the topics that are covered, we have interactive and visual demonstrations that will illustrate rather than explain them. Although we do not go deep into the details of exactly how the implementation of the code looks like, we did leave some links to articles and practice problems to ensure that you really understand the concepts well.
